
### **Installation**

```bash
npm install
npm start
```

### **Provider Configuration**

Edit `src/config/providers.json` to switch between AI models:

```json
{
  "providers": [
    {
      "name": "azure",
      "model": "gpt-4",
      "active": true,
      "maxTokens": 4000,
      "temperature": 0.7
    },
    {
      "name": "google",
      "model": "gemini-pro",
      "active": false,
      "maxTokens": 4000,
      "temperature": 0.7
    }
  ]
}
```
